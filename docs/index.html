⚙️ Installation Steps

Follow these steps in order. The entire process takes under 2 minutes.

⚠️
Do this first — Unblock the file. Windows blocks files downloaded from the internet. You must unblock the .xlam file before adding it to Excel, or macros will be silently disabled.
1
Download the .xlam file
Download ZAF_tool_v3.xlam from the email or from your download page. Save it to your AddIns folder for the smoothest experience:
C:\Users\YourName\AppData\Roaming\Microsoft\AddIns
Tip: Type %AppData%\Microsoft\AddIns directly into Windows Explorer to open this folder instantly.
2
Unblock the file (Windows Security)
Right-click ZAF_tool_v3.xlam → Properties → at the bottom of the General tab, check the "Unblock" box → click Apply → OK.

⚠ This step is required. Skipping it causes macros to be silently blocked — the ZAFtool tab will appear but buttons won't work.
3
Open Excel and go to Add-ins
Open any Excel workbook (or a blank one). Click File → Options → Add-ins.
4
Manage Excel Add-ins
At the bottom of the screen, set the dropdown to "Excel Add-ins" and click Go.
5
Browse and select the file
Click Browse → navigate to where you saved ZAF_tool_v3.xlam → select it → click OK.
6
Enable the add-in
Ensure the checkbox next to "ZAF_tool_v3" is ticked → click OK.
✓
Verify installation
You should now see a "ZAF Tools" tab in your Excel ribbon. If you don't, close Excel completely and reopen it — the tab will appear.

🔐 Enable Macros (Required)

ZAFtool is powered by VBA macros. If macros are blocked, the ribbon will appear but no buttons will work.

💡
Recommended: Use Trusted Locations. Adding the add-in folder as a Trusted Location means Excel will never block ZAFtool macros again — even after updates. Go to Trust Center → Trusted Locations → Add new location.
1
Open Trust Center
File → Options → Trust Center → Trust Center Settings
2
Go to Macro Settings
Select "Macro Settings" from the left panel.
3
Enable VBA macros
Choose "Enable VBA macros" — or "Disable all macros with notification" (recommended if your IT policy is strict). Click OK.
4
Restart Excel
Close Excel fully and reopen it. Your macro settings are now applied.

🔧 Troubleshooting

Problem Solution
ZAF Tools tab not visible Close and reopen Excel fully. Check File → Options → Add-ins → Go and ensure the checkbox is ticked.
"Macros have been disabled" Go to File → Options → Trust Center → Macro Settings and enable VBA macros. Or add the AddIns folder as a Trusted Location.
"This add-in is blocked" Right-click the .xlam file → Properties → check Unblock at the bottom → Apply. Then re-enable in Excel Add-ins.
Licence key not accepted Type the key exactly as provided — no extra spaces. Keys are case-insensitive. Copy-paste directly from the email.
Trial expired early Trial runs 14 calendar days from first activation. Contact abhishek.bhandari@zaftool.com if you need an extension.
Buttons appear but do nothing Macros are likely disabled. Check Trust Center settings. If the issue persists, remove the add-in and re-add it from scratch.

Method 1: Power Query

1

Put all source files in one folder

All files must have the same structure and column headers.

2

Open Power Query

Data → Get Data → From File → From Folder.

3

Combine and Transform

Combine → Combine & Transform Data. Select which sheet to import.

4

Close and Load

Apply filters, then Close & Load to output consolidated data.

5

Refresh next month

Replace files in same folder and click Data → Refresh All.
